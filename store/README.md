## Installation

Must alread have create-react-app installed
To download dependacies for this project run:

```bash
$ npm i react-router-dom express cors stripe
```

To start expressjs server run:

```bash
$ npm start server.js
```

To start reactjs application run:

```bash
$ npm start
```

or

```bash
$ yarn start
```
